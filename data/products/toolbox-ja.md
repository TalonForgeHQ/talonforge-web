## 1. はじめに

AI企業スターターキットツールボックスへようこそ。このガイドに従うことで、2時間で完全に機能するAI企業を構築できます。

### あなたが作るもの

- **自律型エージェント** — 自己修復、積極的、24時間365日稼働
- **コンテンツエンジン** — ウィルス性コンテンツを自動生成
- **SNSマシン** — X、Telegram、Instagram、その他に自動投稿
- **販売システム** — クロウトペイメントによる自動決済
- **セキュリティレイヤー** — 監視、保護、自己修復

### 価値観

私たちは**無料ツール**が連携して機能することを信じています。高額なソフトウェアサブスクリプションはありません。複雑なエンタープライズツールもありません。最高の無料リソースをワイヤードシステムとして統合します。

### 必要なもの

- **コンピューター**: Windows、Mac、またはLinux（どれでも動作）
- **インターネット接続**: 高速推奨
- **Telegramアカウント**: OpenClawを使用するため
- **暗号通貨ウォレット**: 支払い用（metamask.comで作成）

### 必要ないもの

- ❌ プログラミングスキル
- ❌ 高額ソフトウェア
- ❌ 技術的な学位
- ❌ マーケティングの経験
- ❌ ビジネスの経験

すべて自動化によって学びます。

## 2. セクション1: OpenClawをインストールする

OpenClawはあなたのAI企業のオペレーティングシステムです。これはあなたのコンピューターで実行されるコマンドラインインターフェース（CLI）です。インストールするすべてのスキルはOpenClawを介して機能します。

### 手順1.1: OpenClawをダウンロードする

1. [openclaw.ai](https://openclaw.ai)へ移動
2. 「Windows用ダウンロード」をクリック（またはMac/Linux）
3. ZIPファイルをデスクトップに展開

**手に入るもの:**
- OpenClaw実行ファイル
- ドキュメント
- 設定ファイル

### 手順1.2: OpenClawをインストールする（Windows）

1. コマンドプロンプトを開く（Win+R → 「cmd」と入力 → Enter）
2. ダウンロードフォルダに移動：
```
cd %USERPROFILE%\Downloads
```

3. OpenClawインストーラーを実行：
```
setup.exe
```

4. プロンプトに従う：
   - 新しいユーザーアカウントを作成（メール: your@telegram.email、パスワード: strong）
   - これはTelegram統合へのアクセスを提供します

5. インストール完了を待つ（1-2分）

### 手順1.3: OpenClawをインストールする（macOS）

1. ターミナルを開く（Cmd+Space → 「Terminal」と入力 → Enter）

2. Homebrewをインストールしていない場合：
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

3. OpenClawをインストール：
```
brew install openclaw
```

4. OpenClawを初期化：
```
openclaw init
```

5. アカウントを作成するようプロンプトが出ます。

### 手順1.4: OpenClawをインストールする（Linux）

1. ターミナルを開く

2. Node.jsをインストールしていない場合：
```
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

3. OpenClawをグローバルにインストール：
```
npm install -g openclaw
```

4. OpenClawを初期化：
```
openclaw init
```

5. アカウントを作成するようプロンプトが出ます。

### 手順1.5: インストールの検証

1. ターミナルを開く（PowerShellはWindows、TerminalはMac/Linux）

2. OpenClawがインストールされているか確認：
```
openclaw --version
```

3. バージョン番号（例：`1.0.0`以上）が表示されます。

4. OpenClawにログイン：
```
openclaw login
```

5. 認証情報を入力します。

6. ログインしているか確認：
```
openclaw status
```

ユーザー情報と接続状態が表示されます。

### 手順1.6: Telegramとペアリングする

OpenClawはTelegramであなたのAI企業と機能する必要があります。

1. モバイルのTelegramを開く

2. @openclaw_botと新しいチャットを開始

3. 「Get Started」または「Connect Account」を選択

4. Telegramに表示される認証コードをコピー

5. OpenClawにペースト：
```
openclaw pair
```

6. プロンプトで認証コードを入力

7. 成功メッセージが表示されます： "Successfully paired with Telegram!"

## 3. セクション2: 無料スキルをインストールする

OpenClawは「スキル」を使用します — 特定のタスクを実行するモジュラーツールです。これらの13のスキルがあなたのAI企業を構成します。

すべてのスキルは**100%無料**でClawHubから提供されています。サブスクリプションや隠れた料金はありません。

### 手順2.1: 13のスキルを理解する

インストール前に、各スキルが何をするかを理解してください：

1. **openclaw** — コマンドセンター（このスキルですべてのスキルを実行）
2. **web-browsing** — ウェブサイトを閲覧、コンテンツ抽出、ページ要約
3. **browser-automation** — ブラウザをプログラムで制御
4. **x-post-automation** — トレンド検出と共にX/Twitterに自動投稿
5. **automation-content-creator** — ビラーコンテンツから完全なワークフロー
6. **social-media-marketing** — SNS戦略の構築と実行
7. **tweet-writer** — 最適化されたX/Twitterコンテンツ作成
8. **proactive-agent-lite** — 自己修復エージェントで問題を修正
9. **ai-sentinel** — 脅威を監視するセキュリティガード
10. **skill-guard** — セキュリティ脆弱性をスキャン
11. **opentweet-x-poster** — 直接X投稿の自動化
12. **image-vision** — 画像分析、OCR、視覚的推論
13. **qwen-vision** — Alibaba CloudビジュアルAIによる高度な分析

**問題：** スキルが応答しない、または失敗する

**解決策：**
1. OpenClawを再起動：
```
/openclaw restart
```

2. スキルステータスを確認：
```
/openclaw skill status
```

```
/openclaw skill install [skill-name] --force
```

4. インターネット接続を確認
5. OpenClawログを確認：
```
/openclaw logs
```
