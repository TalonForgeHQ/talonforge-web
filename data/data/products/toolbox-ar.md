## 1. مقدمة

مرحباً بك في قاعدة أدوات شركة الذكاء الاصطناعي. من خلال اتباع هذا الدليل، ستكون لديك شركة ذكاء اصطناعي تعمل في حوالي ساعتين.

### ما ستبنيه

- **وكيل تلقائي** — يعالج نفسه، ويكون تحفظياً، ولا ينام أبداً
- **محرك المحتوى** — ينشئ محتوى فيروسي تلقائياً
- **آلة التواصل الاجتماعي** — ينشر إلى X، تيليجرام، إنستغرام، والمزيد
- **نظام المبيعات** — دفعات مع بوابات الدفع الكريبتو
- **طبقة الأمان** — تراقب، تحمي، وتشفي نفسها

### الفلسفة

نحن نؤمن باستخدام **أدوات مجانية** تعمل معاً. لا توجد اشتراكات برامج باهظة الثمن. لا يوجد أنظمة معقدة. فقط أفضل الموارد المجانية في العالم، موصلة في نظام واحد.

### ما تحتاجه

- **جهاز كمبيوتر**: Windows، Mac، أو Linux (يعمل مع أي نظام)
- **اتصال بالإنترنت**: عالي السرعة مفضل
- **حساب تيليجرام**: لاستخدام OpenClaw
- **محفظة كريبتو**: للإشعارات (metamask.com، أنشئ محفظة)

### ما لا تحتاجه

- ❌ مهارات البرمجة
- ❌ برمجيات باهظة الثمن
- ❌ دراسة تقنية
- ❌ خبرة تسويقية
- ❌ خبرة عمل

سنتعلم كل ذلك من خلال الأتمتة.

## 2. القسم 1: تثبيت OpenClaw

OpenClaw هو نظام تشغيل شركة الذكاء الاصطناعي الخاصة بك. إنه واجهة سطر الأوامر (CLI) تعمل على جهاز الكمبيوتر الخاص بك. جميع المهارات التي سنتثبيتها تعمل من خلال OpenClaw.

### خطوة 1.1: تنزيل OpenClaw

1. اذهب إلى [openclaw.ai](https://openclaw.ai)
2. انقر على "تحميل للويندوز" (أو Mac/Linux)
3. استخرج ملف ZIP إلى سطح المكتب الخاص بك

**ما ستحصل عليه:**
- ملف تنفيذي OpenClaw
- مستندات
- ملفات إعدادات

### خطوة 1.2: تثبيت OpenClaw (Windows)

1. افتح موجه الأوامر (Win+R → اكتب "cmd" → Enter)
2. انتقل إلى مجلد التنزيلات:
```
cd %USERPROFILE%\Downloads
```

3. شغل برنامج تثبيت OpenClaw:
```
setup.exe
```

4. اتبع الإرشادات:
   - أنشئ حساب مستخدم جديد (البريد: your@telegram.email، كلمة المرور: strong)
   - هذا يعطيك الوصول إلى التكامل مع تيليجرام

5. انتظر حتى تكتمل التثبيت (1-2 دقيقة)

### خطوة 1.3: تثبيت OpenClaw (macOS)

1. افتح التيرمينال (Cmd+Space → اكتب "Terminal" → Enter)

2. قم بتثبيت Homebrew إذا لم يكن لديك:
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

3. قم بتثبيت OpenClaw:
```
brew install openclaw
```

4. قم بتهيئة OpenClaw:
```
openclaw init
```

5. أنشئ حسابك عندما تُطلب.

### خطوة 1.4: تثبيت OpenClaw (Linux)

1. افتح التيرمينال

2. قم بتثبيت Node.js إذا لم يكن لديك:
```
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

3. قم بتثبيت OpenClaw عالمياً:
```
npm install -g openclaw
```

4. قم بتهيئة OpenClaw:
```
openclaw init
```

5. أنشئ حسابك عندما تُطلب.

### خطوة 1.5: التحقق من التثبيت

1. افتح التيرمينال الخاص بك (PowerShell في Windows، Terminal في Mac/Linux)

2. تحقق مما إذا كان OpenClaw مثبتاً:
```
openclaw --version
```

3. يجب أن ترى رقم إصدار مثل `1.0.0` أو أعلى.

4. سجل الدخول إلى OpenClaw:
```
openclaw login
```

5. أدخل بيانات الاعتماد الخاصة بك.

6. تحقق مما إذا كنت مسجلاً دخولك:
```
openclaw status
```

يجب أن ترى معلومات المستخدم وحالة الاتصال.

### خطوة 1.6: الإقران مع تيليجرام

OpenClaw يحتاج إلى تيليجرام ليعمل مع شركة الذكاء الاصطناعي الخاصة بك.

1. افتح تيليجرام على هاتفك

2. ابدأ محادثة جديدة مع **@openclaw_bot**

3. عندما تُطلب، اختر "ابدأ" أو "ربط الحساب"

4. انسخ كود الترخيص الموجود في تيليجرام

5. الصق الكود في OpenClaw:
```
openclaw pair
```

6. أدخل كود الترخيص عندما تُطلب.

7. يجب أن ترى رسالة نجاح: "تم الربط بنجاح مع تيليجرام!"

### خطوة 1.7: اختبار أولي

1. في OpenClaw، أرسل أمراً بسيطاً:
```
/openclaw hello
```

2. افتح تيليجرام وافحص محادثتك مع @openclaw_bot

3. يجب أن ترى الرد: "مرحباً! أنا OpenClaw. كيف يمكنني مساعدتك في بناء شركة الذكاء الاصطناعي اليوم؟"

## 3. القسم 2: تثبيت المهارات المجانية

OpenClaw يستخدم "المهارات" — أدوات جزئية التي تقوم بمهام محددة. سنتثبيت 13 مهارة التي تشكل شركة الذكاء الاصطناعي الخاصة بك.

جميع المهارات **100٪ مجانية** ومرخصة من ClawHub. لا توجد اشتراكات، ولا رسوم مخفية.

### خطوة 2.1: فهم 13 مهارة

قبل التثبيت، افهم ما تقوم به كل مهارة:

1. **openclaw** — مركز أوامرك (استخدم هذا لتشغيل جميع المهارات)
2. **web-browsing** — تصفح المواقع، استخراج المحتوى، تلخيص الصفحات
3. **browser-automation** — التحكم في المتصفحات برمجياً
4. **x-post-automation** — نشر تلقائي إلى X/Twitter مع اكتشاف الاتجاهات
5. **automation-content-creator** — سير عمل محتوى فيروسي من البداية للنهاية
6. **social-media-marketing** — بناء وتنفيذ استراتيجية التواصل الاجتماعي
7. **tweet-writer** — كتابة محتوى X/Twitter محسّن
8. **proactive-agent-lite** — وكيل يشفي نفسه تلقائياً ويصلح المشاكل
9. **ai-sentinel** — حارس الأمان الذي يراقب التهديدات
10. **skill-guard** — يفحص المهارات للثغرات الأمنية
11. **opentweet-x-poster** — نشر X المباشر تلقائياً
12. **image-vision** — تحليل الصور، OCR، واستنتاج بصري
13. **qwen-vision** — مساعد الرؤية Alibaba Cloud للتحليل المتقدم

### خطوة 2.2: تثبيت المهارات (واحدة تلو الأخرى)

## 5. القسم 4: إعداد Apify

Apify هو محرك اكتشاف المحتوى الخاص بك. يجد محتوى فيروسي على الويب يمكن استخدامه في شركة الذكاء الاصطناعي الخاصة بك.

### خطوة 4.1: إنشاء حساب Apify

1. اذهب إلى [apify.com](https://apify.com)
2. انقر على "ابدأ مجاناً" (Start for free)
3. استخدم البريد الإلكتروني أو Google sign-up
4. تحقق من بريدك الإلكتروني
5. سجل الدخول

## 6. القسم 5: إعداد NowPayments

NowPayments هو بوابة دفع كريبتو تتعامل مع فواتير منتجاتك الرقمية.

### خطوة 6.1: إنشاء حساب NowPayments

1. اذهب إلى [nowpayments.com](https://nowpayments.com)
2. انقر على "Sign Up" (أقصى اليمين)
3. أدخل بريدك الإلكتروني
4. أنشئ كلمة مرور قوية
5. تحقق من بريدك الإلكتروني
6. سجل الدخول
